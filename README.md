# Jsoup_Crawling
Spring Boot 프로젝트에서 Jsoup 라이브러리를 통한 웹스크랩, jpa로 DB에 저장성공 - 1가지 언론사, Rank 5에만 제한, (여러가지언론사는 코드 몇줄 추가해야함) 
추후에 신문기사 관련 프로젝트에서 사용예정
